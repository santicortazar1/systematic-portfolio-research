# Systematic Portfolio Research 

Exploratory systematic portfolio research and macroeconomic regime analysis in Python.

---

## Overview

This repository explores how systematic multifactor portfolio strategies behave across different macroeconomic environments compared to traditional 60/40 allocation models.

The project focuses on risk-adjusted performance, drawdown resilience, and portfolio adaptability using Python-based financial analysis and ETF portfolio construction.

Current research includes the evaluation of Momentum, Low Volatility, and Quality factor strategies during changing market conditions, including periods associated with U.S. Treasury yield curve inversion (10Y–2Y).

---

## Objectives

* Evaluate systematic portfolio allocation approaches using Python
* Compare multifactor strategies against traditional 60/40 portfolios
* Analyze portfolio behavior across changing macroeconomic environments
* Develop reproducible financial research workflows
* Expand analytical and technical implementation capabilities over time

## Research Context

This repository expands on an academically reviewed undergraduate research project originally developed in Spanish and later adapted into an English research and implementation-oriented format.

The broader objective is to progressively evolve the project into a more robust systematic research framework integrating portfolio analytics, macroeconomic analysis, and Python-based financial workflows.

---

## Current Limitations

This project is exploratory and educational in nature.

Current limitations include:

* Simplified portfolio implementation assumptions
* Transaction costs not yet modeled
* Ongoing factor construction and robustness expansion
